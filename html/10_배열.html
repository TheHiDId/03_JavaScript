배열(Array)

    - 변수란?: 메모리 상에 값을 저장하는 공간, 값은 하나만 저장할 수 있음

    - 배열이란?: 변수를 묶음으로 다루는 것

      - 배열에 저장된 요소를 구분하기 위해서 index(색인) 이용 (0부터 시작)

      - JS 배열의 특징
        1. 자료형에 제한이 없음 (공간마다 다른 자료형 저장 가능, 보통은 한 배열에 한가지 자료형으로 통일)
        2. 길이에 제한이 없음 (자동으로 배열의 길이가 조정됨)
        3. 배열을 위한 다양한 필드(변수)/메서드(함수)가 제공됨
          ex) 배열.length -> 배열의 길이 반환
              배열.indexOf("값") -> 값이 몇 번 인덱스 요소에 있는지 반환 (없으면 -1)
  
    [JS 배열 선언법]

    1. const arr = [];: 0칸 짜리 배열을 생성
      - []: 배열 기호

    2. const arr = new Array();: 1번과 같은 기능의 다른 문법
      - new 연산자: 새로운 객체 생성

    3. const arr = new Array(n);: n칸 짜리 배열 생성, 처음 생성될 때에는 n칸 모두 비어있음(empty)

    4. const arr = [값1, 값2, 값3];: 각 요소에 값이 대입되는 3칸 짜리 배열 생성

배열과 for문

    - 배열에 저장된 요소들은 인덱스(index)를 이용해서 구분

    - 인덱스는 0부터 시작해서 배열 길이 -1까지 1씩 증가하는 규칙성을 가짐

      - 인덱스는 무조건 0이상의 정수만 가질 수 있음

    - 인덱스 값을 증가시켜 배열 내 요소를 접근할 때 가장 효율적인 방법이 for문을 이용하는 것

2차원 배열(n차원 배열)

    변수: 메모리 상에 값을 저장하는 공간, 값 1개
      ex) const num1 = 1;
          const num2 = 2;
          const num3 = 3;

    1차원 배열: 변수를 묶음으로 다루는 것, 변수 n개
      ex) const arr1 = [1, 2, 3];
          const arr2 = [4, 5, 6];
          const arr3 = [7, 8, 9];

    2차원 배열: 1차원 배열을 묶음으로 다루는 것, 1차원 배열 n개
      ex) const arr = [
                        [1, 2, 3], 
                        [4, 5, 6], 
                        [7, 8, 9]
                      ]
          arr[0][2] === 3;
          arr[2][1] === 8;

    n차원 배열: n-1차원 배열을 묶음으로 다루는 것
