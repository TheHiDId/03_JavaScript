DOM(Document Object Model)

    Document(HTML) Object(JavaScript) Model

    - HTML 문서를 객체 기반 형식으로 표현한 것
      
      - HTML 문서에 작성된 내용을 트리 구조로 나타낸 객체

DOM을 이용한 요소 접근 방법

    1. id 속성값을 이용한 접근

      - document.getElementById("id 속성값");

      - Document 아래 "id 속성값"이 일치하는 요소를 찾아 반환

      * HTML의 id(식별자) 속성: 중복되면 안 됨(문서 전체에 하나만 존재함)
                                단수(Element) 형태로 작성됨


    2. class 속성값을 이용한 접근

      - document.getElementsByClassName("class 속성값");

      - 클래스가 일치하는 모든 요소를 유사 배열 형태로 반환


    3. tag 이름을 이용한 접근

      - document.getElementsByTagName("tag 이름");

      - 태그 이름이 일치하는 모든 요소를 유사 배열 형태로 반환


    4. name 속성을 이용한 접근
    
      - document.getElementsByName("name 속성값");

      - 네임 속성값이 일치하는 모든 요소를 유사 배열 형태로 반환


    5. CSS 선택자를 이용한 접근

      1. document.querySelector("CSS 선택자");

        - 선택자가 일치하는 요소 중 첫번째 요소만 반환

      2. document.querySelectorAll("CSS 선택자");

        - 선택자가 일치하는 요소를 전부 유사 배열 형태로 반환
