조건문

    - 프로그래밍 코드가 해석되는 순서를 제어하는 "제어문" 중 한 종류

    - 작성된 [조건식]에 따라서 수행되는 코드를 변경하는 구문

    * 조건식 : 결과가 true 또는 false인 식
      ex) 논리 연산자(&&, ||, !)
          비교 연산자(>, ==, != 등)
          함수 호출 결과로 boolean 반환
          boolean 단독 작성

if - else if - else문

    
      if(조건식1) {
        조건식1이 참일 경우 수행할 코드;
      } else if(조건식2) {
        조건식1이 거짓이고, 조건식2가 참일 경우 수행할 코드; 
      } else {
        조건식이 전부 거짓일 경우 수행할 코드;
      }

입력받은 수가 양수인지 확인

1 ~ 100 사이의 난수를 발생시켜 홀/짝 판별

양수, 음수, 0 판별

어린이, 청소년, 성인 구분

    0 ~ 13: 어린이
    14 ~ 19: 청소년
    20 ~ 120: 성인
    0미만, 120초과: 잘못 입력 하셨습니다.
